# API skypattern

API skypattern

## Comment lancer l'application?

### 0 : Installer node

#### Windows

https://nodejs.org/en/download/

#### Ubuntu / Debian

sudo apt install npm

### 1 : Installer les dépendances

Required: JAVA 11

npm install express --save

### 2 : Lancer le serveur

npm start
